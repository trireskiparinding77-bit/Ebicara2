Pengaturan

Kunci API gratis dari Google AI Studio. Disimpan hanya di browser kamu sendiri (localStorage), tidak dikirim ke server manapun selain Google. Dapatkan di aistudio.google.com/apikey.

Kecepatan bicara AI lebih pelan

eBicara berjalan sepenuhnya di browser kamu. Pengenalan suara pakai fitur bawaan Chrome, jadi pastikan pakai browser Chrome/Edge di desktop atau Android untuk hasil terbaik. Tidak ada data percakapan yang dikirim ke server eBicara — hanya ke Gemini API milik Google memakai API key kamu sendiri.
